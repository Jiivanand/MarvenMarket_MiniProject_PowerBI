# Maven Market Power BI Dashboard

## Project Overview
This project showcases an interactive **Power BI dashboard** developed for **Maven Market** to analyze sales performance, profitability, returns, and geographic trends.  
The dashboard enables data-driven decision-making by providing clear insights into key business metrics.

---

## Objectives
- Track **current month transactions, profit, and returns**
- Analyze **product brand performance**
- Monitor **profit margins and return rates**
- Identify **weekly revenue trends**
- Visualize **geographic sales distribution**
- Compare **actual revenue vs target**

## Tools & Technologies Used
- Power BI Desktop  
- Power Query (Data Cleaning & Transformation)  
- DAX (Calculated Measures & KPIs)  
- Microsoft Bing Maps (Geospatial Analysis)

---

## Data Preparation
- Cleaned and transformed raw data using Power Query  
- Handled missing values and formatting issues  
- Created calculated columns and measures for:
  - Profit
  - Profit Margin
  - Return Rate
  - Month-over-Month Growth

## Dashboard Highlights
- Fully interactive visuals with slicers and filters  
- Business-friendly and professional layout  
- Performance-based color formatting  
- Designed for executive and managerial analysis

## How to Use
1. Download the `PowerBI_MarvenMarket.pbix` file  
2. Open it in **Power BI Desktop**  
3. Refresh the data if required  
4. Use filters and slicers to explore insights
